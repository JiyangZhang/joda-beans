
Joda-Beans
================================================
Joda-Beans is a library that provides full bean and property support for the JDK.

This is release 0.7 of Joda-Beans (Alpha).

The release depends on Java SE 6.

See http://joda-beans.sourceforge.net/changes-report.html for changes
